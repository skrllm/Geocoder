Для запуска: Необходимо открыть проект через Visual Studio, а также иметь .net Framework версии не менее 4.7.2.

Описание:
Приложение построено по паттерну MVVM с использованием карт BING.
Существует главное окно MainWindow.xaml, В котором расположен весь Фронтенд.
Все элементы привязаны к MapViewModel.
Запрос к серверам Microsoft не работают без индивидуальных ключей разработчиков, поэтому данный ключ был получен и внедрен в MapModel.cs


Важно: 
1) Для работы необходимо иметь Интернет подключение.

Процесс работы: 
1) Слева в поле печатаем пунк, который необходимо найти.
2) Нажимаем на кнопку найти.
3) Если местоположение найдено, то на карте появляется найденная точка, а в истории добавится элемент.
4) Если местополодение не найдено, приложение сообщит об этом в строке под кнопкой.



Описание работы приложения:
1) Приложение принимает текст поиска
2) Приложение формирует запрос и отправляет его на сервера Microsoft.
3) Приложение получает ответ.
4) Если локация по адресу найдена, то появляется точка на карте, а текст запроса остается в истории
5) Если локация не найдена или запрос пуст, появляется текст ошибки
